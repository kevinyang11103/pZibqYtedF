## 项目概述 java163

本项目是一款社团管理系统，基于B/S架构和MVC模式开发，采用前后端分离的技术方案。系统主要由Java、SpringBoot、Vue等技术栈构建，支持多角色操作，致力于提高社团管理的效率和便捷性。

## 技术栈

- **架构**：B/S、MVC
- **系统环境**：Windows
- **开发环境**：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 8.0
- **技术栈**：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

系统包含以下三个角色：

- **用户**：普通登录用户，可以进行注册和用户信息管理。
- **超级管理员**：拥有所有社团管理的权限。
- **社团管理员**：负责本社团的信息管理、活动管理、入团申请处理及费用管理。

### 功能列表

- **用户模块**：
  - 登录/注册
  - 用户信息管理

- **超级管理员**：
  - 所有社团管理

- **社团管理员**：
  - 社团信息管理
  - 社团活动管理
  - 入团申请处理
  - 费用管理
  - 社团活动通知

- **学生用户**：
  - 社团信息查看
  - 社团活动参与
  - 入团申请
  - 费用缴纳
